## 通用项目模版

总体来说，大型项目不会把所有代码放在一个 py 文件中，通常要分好几个模块。并且需要与外部文件读写。所以涉及模块管理，文件管理两方面。

模块管理：
要有一个主程序。
config

其他文件管理：
- assets 目录：存放与不与程序打交道的各种图片等资源。例如，README.md 文件的图片。
