# 水饺 / 包子 / 馅饼
煮饺子方法
1. 烧水，等待开锅
2. 下饺子：一堆一堆地下，下一部分搅一下，避免粘锅
3. 盖上盖，过一小会用筷子搅一下，等待开锅
4. 开锅，盖上盖，如果要窜了，开盖（盖竖起来），往里搅，中间可以点水，点三次
饺子都浮起来，关小火，捞饺子，注意控水快捞，捞完关火
